# Remos Internship

Some of the code, research, data, and hardware debugging from my time at Remos

## Repository Overview
| Directory | Description |
|---|---|
| `hpa-calibration` | HPA Calibration Tool suite (including Modbus device tools + wiring) |
| `research` | A look at Voltage Standing Wave Ratio hardware/software, and an overview of SDR and RF concepts |
| `rohde-and-schwarz-fsh6` | Service manual and some documentation of the mainboard PCB issues |
| `sequencer-measurements` | Data points measured during development for calibrating an HPA |
